Cashier App to calculate the change on a Parking Lot
Input the amount and value of bills available
Input the hour rate and get the total amount to be paid
Input the amount and values of the money paid
The machine will calculate the change using as few bills as possible

NOTE: client can only pay on bills specified on stock (if you need a particular value of bill but dont have it just add it with an amount of 0 at the start)
working link: https://parking-app-mdp.netlify.app/
